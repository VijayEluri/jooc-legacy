Chapter 2. Functions

Syntax

Basic syntax

A function is declared with the help of the func keyword.

// Empty function with no args that returns nothing
func useless;

func sayHello {
  printf("Hay, world!\n");
}

func saySomethingElse(String message) {
  printf("Dummy says: %s\n", message);
}

func add(Int arg1, Int arg2) -> Int {
  return arg1 + arg2;
}
