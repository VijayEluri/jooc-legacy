Callbacks and pointers

Simple case

In ooc, functions are almost-first-class-citizens. You can pass pointer to functions around easily:

func main {

  call(@kotokoSpeak);

}

func kotokoSpeak {

  printf("We have a very big problem!\n");

}

func call(Func toCall) {

  toCall();

}
